📖 What is Verilog HDL?

Hardware Description Language used to model electronic systems. Not a programming language — it describes parallel hardware behavior. IEEE 1364 standard. Supports RTL, gate-level, behavioral modeling.

⚙️ Digital Verilog

Describes digital circuits: combinational (mux, decoder, ALU) and sequential (flip-flops, counters, registers, FSMs). Used for synthesis (netlist), simulation, timing analysis.

🚀 Real‑world Applications

  • ASIC / FPGA (CPU, GPU, DSP)
  • Automotive: ADAS, infotainment
  • Communication: UART, SPI, PCIe
  • Consumer electronics: IoT, camera sensors
  • RISC‑V / ARM cores

🏗️ Types of Verilog Structure

  • Modules / ports
  • Continuous assignments (assign)
  • Procedural blocks (always, initial)
  • Gate‑level & dataflow
  • Finite State Machines (Mealy/Moore)
  • Testbenches (timing, monitors)
